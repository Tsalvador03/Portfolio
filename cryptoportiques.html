Cryptoportiques d'Arles

2024 — SAE
Unity TypeScript HTML MySQL Groupe de 4

Vue d'ensemble

Visite virtuelle des cryptoportiques romains d'Arles, réalisée pour la mairie dans le cadre d'une SAE. Le projet combine une application Unity pour la visite 3D et un back-office web pour la gestion des sessions de jeu.

Fonctionnalités

  • Visite virtuelle en 3D des cryptoportiques
  • Points d'intérêt interactifs
  • Back-office de gestion du contenu
  • Base de données pour les sessions de jeu

Stack technique

  • Unity — moteur 3D / visite virtuelle
  • TypeScript — back-end API
  • React/CSS — interface web
  • MySQL — base de données

Mon rôle

  • Développement de la partie Unity
  • Développement du premier backend

Contexte

  • Projet commandé par la mairie d'Arles
  • SAE — situation d'apprentissage et d'évaluation
  • Équipe de 4 étudiants

Bilan

Ce projet m'a permis de travailler sur un vrai cas client avec une commande concrète. Jongler entre Unity et une stack web complète (TypeScript + MySQL) en équipe était ambitieux, mais très formateur. Les présentations à la mairie était de premières expérience de présentations d'un produit en conditions réelles.
