Portfolio / Projet 01

Gradely

React Native TypeScript SQLite Approche Agile
2023 — 2024

Vue d'ensemble

Application mobile permettant à un professeur de noter des soutenances et de générer des grilles de notation personnalisées.

Objectifs du projet

  • Interface intuitive pour la gestion des notes
  • Système de création de grilles de notation
  • Fonctionnalités de partage et d'export

Technologies utilisées

  • React Native — développement cross-platform
  • TypeScript — typage statique
  • SQLite — base de données locale
  • Bun — runtime & bundler
  • Figma — design UI/UX
  • Git — versioning

Mon rôle

  • Architecture de l'application
  • Développement des fonctionnalités principales
  • Intégration avec Bun
  • Coordination avec l'équipe de design

Méthode de travail

  • Sprints hebdomadaires (Agile/Scrum)
  • Revues de code régulières
  • Tests utilisateurs avec des enseignants

Galerie

Bilan

Ce projet m'a permis d'approfondir mes compétences en développement mobile avec React Native et d'expérimenter la gestion de projet en méthode agile. J'ai particulièrement apprécié l'aspect design centré sur l'utilisateur et la coordination en équipe sur plusieurs mois.
